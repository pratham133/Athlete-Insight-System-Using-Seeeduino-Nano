# Athlete-Insight-System-Using-Seeeduino-Nano
The Athlete Insight System is a low-cost, open-source wearable that tracks heart rate, SpO2, and activity in real-time. It uses a Seeeduino Nano, BMA456, and MAX30102 sensors, costing under $15. Data is sent to an Android app for easy viewing. This project provides a simple, affordable, and effective option compared to expensive fitness trackers.
